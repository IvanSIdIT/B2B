# Factory Line Manual

## Conveyor belt (Line A)

If the conveyor belt stops unexpectedly, check the emergency stop button near station 3.
Reset the button by twisting it clockwise until it clicks.

## Hydraulic press

Normal operating pressure is 120–140 bar. If pressure drops below 100 bar, shut down the press
and report the issue to the shift manager immediately.

## Lubrication schedule

Bearings on Line A must be lubricated every 40 operating hours. Use ISO VG 68 oil only.
